# digit-converter-multilang

Convert numbers into digits of different languages (Hindi, Gujarati, Arabic, Persian, etc.).

## ЁЯУж Installation

```bash
npm install @dmxdev/digit-converter-multilang
```

## ЁЯЪА Usage

```bash
import { convertDigits } from "digit-converter-multilang";

console.log(convertDigits(1234567890, "hi")); // резреирейрекрелремренреорепреж
```

## ЁЯМР Supported Languages

- English (en)
- Hindi (hi)
- Marathi (mr)
- Sanskrit (sa)
- Gujarati (gu)
- Bengali (bn)
- Tamil (ta)
- Telugu (te)
- Kannada (kn)
- Malayalam (ml)
- Punjabi (pa)
- Arabic (ar)
- Persian (fa)
- Urdu (ur)
